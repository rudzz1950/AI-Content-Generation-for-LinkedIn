2. NOTION INTEGRATION (Post Storage)
================================================================================

Store your generated LinkedIn posts in Notion for review and archival.

Your Database: https://cosmic-bone-56d.notion.site/LinkedIn-Posts-2ecb15cc6736806e9dd7f174b4a1dbe0

Setup Steps:
1. Go to https://notion.so/my-integrations
2. Click "New Integration" → Name it "Weekly Article Flow"
3. Copy the "Internal Integration Secret" (this is your NOTION_API_TOKEN)
4. Open your LinkedIn Posts database in Notion
5. Click Share → Add connections → Select your integration
6. Copy Database ID from the URL (the long string before ?v=)

4. SAMPLE .env FILE
================================================================================

Copy this template and fill in your values:

------- START OF .env FILE -------

# API Keys (Required)
GEMINI_API_KEY=your_gemini_key_here
PERPLEXITY_API_KEY=your_perplexity_key_here
GROQ_API_KEY=your_groq_key_here
LANGSMITH_API_KEY=your_langsmith_key_here
NOTION_API_TOKEN=your_notion_token_here

# Configuration
NOTION_DATABASE_ID=your_notion_db_id_here
ENABLE_MEDIUM_BYPASS=false

# User Preferences
PREFERRED_TOPICS=LLMs,AI Agents,Startups
EXCLUDED_TOPICS=Crypto,NFT
POST_STYLE=professional

# LangSmith Tracing
LANGCHAIN_TRACING_V2=true
LANGCHAIN_PROJECT=weekly-article-flow

------- END OF .env FILE -------
